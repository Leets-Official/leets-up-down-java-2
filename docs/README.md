## 기능 요구사항

### 객체별 역할

**View**
- [x] 게임 버전(영어/숫자)입력 받기
- [x] 숫자 입력받기
- [x] 영어 입력받기

**Output**
- [x] 게임 시작 문구 출력
- [x] UP/DOWN 결과 출력
- [x] 정답이면 정답문구 출력
- [x] 시도환 횟수 출력

**Controller**
- [x] 게임 시작
- [ ] 입력값 유효성 검사

**Model**
- [x] 랜덤수 생성
- [x] 랜덤수보다 입력값이 크면 UP 판단
- [x] 랜덤수보다 입력값이 작으면 DOWN 판단
- [x] 랜덤수가 입력값과 같으면 게임 종료
- [x] 시도횟수 판단

**에러처리**
- [x] 버전 이외의 값 입력
    :[ERROR] 존재하지 않는 버전입니다.
- [ ] 범위 밖의 숫자 입력
    :[ERROR] 범위 내의 숫자를 입력하세요.
- [ ] 범위 밖의 알파벳 입력
    :[ERROR] 범위 내의 알파벳을 입력하세요.
- [ ] 다른 타입의 입력값 입력
    :[ERROR] 입력 문자의 타입이 맞지 않습니다.
